隱私政策

最後更新日期：2026年6月23日

簡介

BudgetPlus 尊重並保護所有用戶的個人隱私。本應用程式致力於確保用戶資料的安全性與私密性。

資料儲存

所有用戶資料（包括帳戶、交易記錄、預算、儲蓄目標、標籤及定期記帳設定）僅儲存於用戶的本機裝置。我們不會將任何資料上傳至雲端伺服器或第三方服務供應商。

權限使用

• 相機權限：僅用於拍攝收據照片

• 相簿權限：僅用於選擇已拍攝的收據圖片

• 通知權限：僅用於每日提醒功能

第三方服務

本應用程式使用 Frankfurter API 獲取即時匯率。該服務僅傳送幣種查詢請求，不包含任何可識別個人身份的資料。

資料分享

我們不會與任何第三方機構、公司或組織分享用戶的個人資料或財務資料。

資料匯出

用戶可隨時透過本應用程式的匯出功能，將個人資料以 CSV 或 PDF 格式匯出至本機裝置。

聯絡我們

如對本隱私政策有任何疑問，請透過 App 內的回饋功能與我們聯繫。
